# Notes Application
## Library Used
#### Material Design
#### Room Database

## Application Overview
Designed to take notes and add your tasks to personalize your tasks and to keep track of them. <br> 
You can add new task and notes.<br>
You can delete them.<br>
You can delete all the task or notes at once.<br>
You can search for items in your list.
